# Design System Document

This document outlines the core aesthetic and functional principles of our design system, as defined by the underlying design tokens.

## Theme Overview

Our current theme emphasizes a **dark** color mode, providing a modern and focused user experience. The color palette is built around a vibrant seed color, aiming for a lively and energetic feel.

## Color Palette

-   **Primary Color:** `#00FF9C` (A vibrant green, used for primary actions and key brand elements)
-   **Neutral Color:** `#0E1111` (A very dark almost black, serving as the base for backgrounds and surfaces)

*(Note: `secondary_color_hex` and `tertiary_color_hex` are not explicitly defined in this theme, implying derivation or a simpler palette.)*

## Typography

The design system utilizes a consistent typeface across all elements:

-   **Headlines:** `Space Grotesk` - Bold and modern for prominent text.
-   **Body Text:** `Space Grotesk` - Clear and readable for general content.
-   **Labels:** `Space Grotesk` - Concise for UI labels and interactive elements.

## Shape and Form

**Roundedness:** `3` (Maximum, pill-shaped)
User interface elements feature a maximum level of roundedness, resulting in soft, pill-shaped corners that convey a friendly and approachable aesthetic.

## Spacing

**Spacing:** `2` (Normal)
The system employs a normal level of spacing, providing a balanced and comfortable density for UI elements and content, ensuring good readability without feeling too sparse or cramped.
